7. Lists

There are three kinds of list:

  • Unordered
  • Ordered
  • Description

7.1 Unordered List

<ul> is used to start an unordered list. Each item should start with <li>. Example:

  • HTML
  • CSS
  • JAVASCRIPT

7.2 Ordered List

<ol> is used to start an ordered list. Each item should start with <li>.
This will display a numbered list. Example:

  1. Red
  2. Orange
  3. Yellow

7.3 Description List

<dl> is used to start an ordered list. Each item should start with title, represented by <dt>.
and <dd> for the description. Example:

COMP 005
- Discrete Structures 2
COMP 006
- Data Structures and Algorithms
