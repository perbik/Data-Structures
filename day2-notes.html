MORE HTML ELEMENTS

HTML Formatting Elements:

1. Bold

<b> is used to bold texts. this has no meaning.

2. Strong

<strong> is used to make text bold, but unlike bold, this element carries meaning which is useful for accessibility and SEO

3. Italic

<i> is for italic and visual styling only

4. Emphasize

<em> is for emphasizing text, useful for accessibility and SEO

5. Mark

<mark> is used to highlight texts.

6. Small

<small> is for making texts smaller in size.

7. Delete

<del> is for deleting texts, however, this will only display a line on the text or strikethrough style

8. Insert

<ins> is used to represent a text that has been added or inserted. This will display an underline

9. Subscript

<sub> is for subscriptssub

10. Superscript

<sup> is for superscriptssup
